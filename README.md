# homebrew-tap
Homebrew Tap for Zasper

### Install [zasper](https://github.com/zasper-io/zasper) using `homebrew`

```bash
brew install zasper-io/tap/zasper
```

### Upgrade `zasper`

```bash
brew update && brew upgrade zasper-io/tap/zasper
```

### Adding the Tap

```bash
brew tap zasper-io/tap
```
