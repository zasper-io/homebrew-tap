# homebrew-tap
Homebrew Tap for Zasper


```bash
brew tap zasper/tap
```

Install [zasper](https://github.com/zasper-io/zasper) using `homebrew`

```bash
brew install zasper/tap/zasper
```

Upgrade `zasper`

```bash
brew update && brew upgrade zasper/tap/zasper
```
